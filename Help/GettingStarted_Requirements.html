Requirements

In order to run the applications of the Gab Encoding Tools suite, you have to meet the following requirements :

  • Microsoft Windows operating system (*)
  • The Microsoft .NET Framework 2.0 must be installed (*)
  • Less than 1 MB on the hard drive for installation
  • 20 MB of RAM to run the program, about 64 MB of free RAM to do some encoding conversions (more or less, depending of the size of your documents)

(*) There is some plans to convert Gab Encoding Tools to Mono, the open-source .NET environment. It will then be available to any platform who can run Mono applications.
