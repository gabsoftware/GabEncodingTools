Installation

There is no need to install Gab Encoding Tools. Just extract the content of the zip archive to any directory : the Gab Encoding Tools suite is now installed.

Notes :

  1. Gab Encoding Tools is provided with some translations. If you want to run the English version of the applications, simply delete or rename any directory that is not "en" (for example delete or rename the directory "fr" if you are French but wish to run the English version of Gab Encoding Tools).
  2. Gab Encoding Converter may create a file to store its settings. It is usually located in the %APPDATA%\GabSoftware folder.
