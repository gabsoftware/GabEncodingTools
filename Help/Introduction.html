Introduction

Gab Encoding Tools is a useful application specificaly designed for converting files of different encodings very easily.

Gab Encoding Tools provide two applications : Gab Encoding Converter and Gab Encoding Converter Console.

Gab Encoding Converter

Gab Encoding Converter is the part of Gab Encoding Tools which provides a convenient and intuitive user interface. It features a two panes interface with on the left the source file and on the right the destination file. The encoding of the source file is automatically detected but can be changed, with a live preview. The encoding of the destination file can be changed and features a live preview also.

Gab Encoding Converter Console

Gab Encoding Converter Console is the console equivalent of Gab Encoding Converter and will be very useful for batch processing.
