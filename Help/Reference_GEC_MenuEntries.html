Gab Encoding Converter menu entries

The menu is very simple and should not cause any problem to someone used to work in the Windows environment.

File menu

Here is a detailed description of the File menu :

  • New (CTRL+N) : Creates a new empty document.
  • Open (CTRL+O) : Opens an existing file.
  • Save (CTRL+S) : Saves the destination file (1).
  • Save As : Saves the destination file under a new filename.
  • Close (ALT+X) : Closes the current document and ask to save it if it has been modified since its last save.
  • Exit (ALT+F4) : Exits Gab Encoding Converter.

Edit menu

Here is a detailed description of the Edit menu :

  • Undo (CTRL+Z) : Undoes the last change (2).
  • Redo (CTRL+Y) : Redoes the last undone modification (2).
  • Copy (CTRL+C) : Copies the selected text into the clipboard.
  • Cut (CTRL+X) : Cuts the selected text into the clipboard.
  • Paste (CTRL+V) : Pastes the content of the clipboard.
  • Find (CTRL+F) : Display the Find window (3).
  • Replace (CTRL+R) : Displays the Replace window (3).

Options menu

Here is a detailed description of the Options menu :

  • Language : This menu allows you to change the language of the application (4).

Favorites menu

Here is a detailed description of the Favorites menu :

  • Add Current Encoding Pair To Favorites : As it says, it adds the current encoding pair to the favorites...
  • Delete Favorites : Deletes ALL the favorites.
  • Edit Favorites : Displays a textbox allowing to quickly edit the favorites. Reserved to advanced users. Format is : SourceCodePage1:DestinationCodePage1;SourceCodePage2:DestinationCodePage2;...
  • Following menu entries : The favorites. Clicking on a favorite will select the appropriate pair of encodings.

Window menu

Here is a detailed description of the Window menu :

  • Entries of this menu : Allow to select the related opened document.

Help menu

Here is a detailed description of the Help menu :

  • Contents (F1) : Displays the current help file.
  • About : Displays the About window which shows the application version.

(1) : By default, the source file is overwritten with the destination file and a backup of the source file is created with the .old extension.

(2) : Currently, only the very last modification can be undone/redone.

(3) : If none of the source file or destination file textbox got the focus, the Source file textbox will be used by default.

(4) : The document window will need to be closed and reopened to be translated in the choosen language.
